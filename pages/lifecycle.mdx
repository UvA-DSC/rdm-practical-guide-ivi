RDM Lifecycle
===

import { Steps } from 'nextra/components'

<Steps>

### Before project

- Plan by writing a [DMP](lifecycle/before/dmp)
- Assess data protection impact (if applicable)
- Complete ethics review (if applicable)

### During project

- Handle [personal data](lifecycle/during/personal) responsibly
- Use appropriate [tools](lifecycle/during) for
  - Data transfer
  - Data processing
  - Data storage

### After project

- Choose a [license](lifecycle/after/license)
- [Archive and publish](lifecycle/after/archive) your work
- [Register](lifecycle/after/registration) your work in Pure

</Steps>
