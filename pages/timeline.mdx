RDM Timeline
===

import { Steps } from 'nextra/components'

<Steps>

### Before project

- Plan by writing a [DMP](timeline/before/dmp)
- Perform [data protection impact assessment (DPIA)](https://medewerker.uva.nl/shared-content-secured/medewerkersites/uva-medewerkers/en/az/privacy/risk-analysis-isp-and-dpia/risk-analysis.html) (if applicable)
- Complete [ethics check](https://medewerker.uva.nl/en/science/content-secured/az/ethics-committees-faculty-of-science/faculty-of-science-ethics-committee-fec.html) (if applicable)

### During project

- Handle [personal data](timeline/during/personal) responsibly
- Use appropriate [tools](timeline/during) for
  - Data transfer
  - Data processing
  - [Data storage](https://xd.adobe.com/view/b42f8e64-0b02-49a6-90fe-7971d4b7f1e3-e17c/)

### After project

- Choose a [license](timeline/after/license)
- [Archive and publish](timeline/after/archive) your work
- [Register](timeline/after/registration) your work in Pure

</Steps>
